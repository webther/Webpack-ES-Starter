# Webpack + ES Starter

#### Features

* Webpack 4
* ES6
* Babel
* ESLint
* SASS
* Unit Test (Jest)
* E2E Test (Cypress)
* Lodash 4

# How to use?

#### Setup local environment.
```
$ npm install
```

#### Run local dev server.
```
$ npm run dev
```

#### Run Unit Test.
```
$ npm run test:unit
```

#### Run E2E Test.
```
$ npm run test:e2e
```

#### Build production.
```
$ npm run build
```
